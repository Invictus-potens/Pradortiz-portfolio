Objetivo do Projeto

O objetivo deste projeto é desenvolver um companheiro inteligente, assistido por Inteligência Artificial, voltado para fazendeiros e agricultores. A solução busca centralizar dados climáticos regionais, oferecer suporte preditivo e auxiliar nas tomadas de decisão relacionadas às atividades agrícolas.

1.1 Motivação

O aumento populacional e mudanças climaticas exigem que a agricultura se torne mais eficiente, fazendeiros sempre buscam soluções que possam aumentar a qualidade de plantio e retorno de investimentos. acreditamos que o uso de inteligência artificial como ferramenta pode ser um diferencial de mercado para pequenos e médios agricultores, fornecendo insights valiosos e suporte em tempo real para otimizar suas operações agrícolas. Com baixa janela de entrada e foco em usabilidade, o I.Agro visa democratizar o acesso a tecnologias avançadas, tornando-as acessíveis e úteis para aqueles que mais precisam.

1.2 Público-Alvo

Pequenos e médios agricultores, cooperativas, hobbystas e etc.

Funcionalidades Principais

2.1 Monitoramento Climático

  • Previsões meteorológicas para a região de interesse
  • Análise de tendências climáticas de curto, médio e longo prazo
  • Alertas automáticos para eventos climáticos extremos

2.2 Suporte à Tomada de Decisão

  • Recomendações de irrigação baseadas em dados climáticos e de solo
  • Controle e prevenção de pragas com base em condições ambientais
  • Orientações sobre manejo da plantação

2.3 Dashboard de Monitoramento

  • Exibição de dados em tempo real
  • Histórico de monitoramento disponível para consulta
  • Gráficos de série temporal (time series)
  • Heatmaps de monitoramento regional (sujeito à disponibilidade de dados)

2.4 Interface de Usuário com Chatbot

  • Interface web integrada ao dashboard
  • Chatbot para interação em linguagem natural
  • Consultas personalizadas aos dados coletados
  • Recebimento de sugestões e recomendações sob demanda

Interface do Projeto

7.1 Chat IA

Aqui é onde podemos interagir com a IA, cada botão em cima da caixa de dialogo inicia uma função que executa querys personalizadas no banco de dados, a IA processa os dados e retorna uma resposta em linguagem natural, o usuário pode fazer perguntas como "Qual a previsão do tempo para os próximos 3 dias?" ou "Qual a umidade atual na minha região?" e receber respostas detalhadas com base nos dados coletados.

7.2 Monitoramento

Aqui é onde podemos visualizar os dados do sistema em tempo real, com gráficos e métricas principais que permitem acompanhar tendências climaticas das regiões fazem parte da busca. Acompanhamento do clima em tempo real, previsão de até 3 dias futuros, nivel de raios UV, humidade e tentencia de chuva são algumas das informações disponíveis no dashboard, o monitoramento é atualizado automaticamente a cada nova coleta de dados realizada pelo n8n, garantindo que os usuários tenham acesso às informações mais recentes para tomar decisões informadas sobre suas atividades agrícolas.
